 orded list is written like <ol> (start tag) <li> (each list item starts with this tag)

HTML for tables

start tags 
 table => <table>
          table row => <tr>
          table header => <thead>
          table header cell => <th>
          table body => <tbody>
          table cell => <td>
          table footer => <tfoot>
          table footer cell => <tfoot>

          code example:
          <table style="width:100%">
            <tr>
            <th>Person 1</th>
            <th>Person 2</th>
            <th>Person 3</th>
            </tr>
            <tr>
            <td>Emil</td>
            <td>Tobias</td>
            <td>Linus</td>
            </tr>
            <tr>
            <td>16</td>
            <td>14</td>
            <td>10</td>
            </tr>
            </table>
